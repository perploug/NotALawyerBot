# notalawyerbot

> A GitHub App built with [Probot](https://github.com/probot/probot) that Github bot to scan for dependency licenses in pull requests

## Setup

```sh
# Install dependencies
npm install

# Run typescript
npm run build

# Run the bot
npm start
```
